# JDBC Banking System

A Java-based console application that demonstrates core banking operations using JDBC and MySQL.

## Features
- Account creation
- Deposit and withdrawal transactions
- Balance enquiry
- Secure database connectivity using JDBC

## Technologies Used
- Java
- JDBC
- MySQL
- IntelliJ IDEA

## How to Run
1. Create a MySQL database and required tables
2. Update the database credentials in the source code
3. Compile and run the main Java class

## Learning Outcome
This project was developed to gain hands-on experience with JDBC, SQL queries, and database connectivity in Java.
